\section{实验介绍}

\subsection{基本信息}

本次实验的内容是测量 $^{12}\text{C}(^{6}\text{Li},^{7}\text{Li})^{11}\text{C}$ 反应中 $^{11}\text{C}$ 的一些激发态的反应截面，其中包括：

\begin{enumerate}
    \item \textbf{第三个 $3/2^{-}$ 态}：这个激发态可能是 Hoyle 态（$^{12}\text{C}$, 7.65 MeV）的类似态。
    
    \item \textbf{$1/2^{-}$， T=3/2 ，12.51 MeV态 }：这个激发态与$^{11}\text{N}$第一激发态、$^{11}\text{Be}$第一激发态一同组成A=11,T=3/2，$J^\pi = 1/2^-$的同位旋多重态。

    \item \textbf{可能存在的$1/2^{+}$， T=3/2 态 }：这个态与基态$^{11}\text{B}$组成同位旋多重态。
    
\end{enumerate}


\subsection{实验背景}

这里需要根据不同的三个目标分别介绍其背景:

\begin{enumerate}
    \item \textbf{第三个 $3/2^{-}$ 态}：Hoyle 态 是$^{12}\text{C}$的一个非常特别的激发态（7.65 MeV），在这个态下，$^{12}\text{C}$中的三个$\alpha$粒子以Borromean环的形式连接，形成一个窄的共振态，这解释了核天体中由$\alpha$粒子合成$^{12}\text{C}$的一个重要途径。在观察到Hoyle 态之后，人们想知道，如果$^{12}\text{C}$失去一个中子或者质子，那这种特殊的结构是否还可能存在。根据理论计算\cite{kanada2007negative}，$^{11}\text{C}$存在一个$3/2^{-}$态，其密度分布呈现出“稀疏的2$\alpha$+$^{3}\text{He}$”结构。 在一些实验中\cite{yamaguchi2013alpha}，则发现在8-10MeV之间有许多共振能级，其中8.1 MeV能级可能就是第三个 $3/2^{-}$ 态。而这个态也确实有一个比较稀疏的质量分布，符合Hoyle 态的特征。这个反应的预期截面与基态的截面在一个数量级，也就是$3×10^{−2}$到 3 mb/sr左右(50 MeV，80度)\cite{kerr1997analyzing}。
    
    \item \textbf{$1/2^{-}$， T=3/2 ，12.51 MeV态 }：同位旋多重态指的是拥有相同质量、自旋宇称、同位旋，只有同位旋第三分量不同的几个核。这些核的核力分布有很高的相似性，在许多性质上表现相似。这些同位旋多重态的激发能遵循库伦位移定律（一个经验公式，对于A=11，T=3/2体系吻合得很好），我们可以根据这个公式猜测各个同位旋多重态对应的能级。$^{11}\text{C}$的这个激发态与$^{11}\text{N}$第一激发态、$^{11}\text{Be}$第一激发态一同组成A=11,T=3/2，$J^\pi = 1/2^-$的同位旋多重态。其中，$^{11}\text{Be}$第一激发态被观察到有中子晕，并且与其对应的$^{11}\text{N}$则观察到了质子晕，形成一对镜像晕核。可以看出这个同位旋多重态的结构特性非常明显，所以我们期待可以在$^{11}\text{C}$的这个激发态也可以观察到比$^{11}\text{Be}$第一激发态更弱一些的类似晕的结构。

    \item \textbf{可能存在的$1/2^{+}$， T=3/2 态 }：这个态与基态$^{11}\text{B}$组成同位旋多重态。寻找这个态可以帮助我们验证库伦位移定律，确认这个态是否是弱束缚或者是有晕的结构。




\end{enumerate}
    有一个难点在于，不仅$^{11}\text{C}$有众多的激发态，$^{7}\text{Li}$也有，尤其是有0.478 MeV这个能量较低的激发态，导致能量谱上可能会有非常丰富的内容。




\subsection{实验束流}
 
这次实验束流由北京串列加速器提供。实验设计上希望可以达到10-15 MeV每核子的能量。根据程序计算，最大剥离概率的电荷态为3+，如果能量要达到1 0MeV每核子，需要串列高压达到15 MV左右。但实际上串列可能只能达到10-12 MV的高压，所以总能量大概是不到50 MeV，截面可能会比较小，实际情况以具体调束为准。

% 这次实验束流由北京串列加速器的二厅Q3D管道进行，使用Q3D磁谱仪对出射粒子进行聚焦和散焦，随后通过焦面探测器对粒子位置进行测量以获得高分辨的动量信息。


\begin{enumerate}
    \item \textbf{第三个 $3/2^{-}$ 态}这个反应的预期截面与基态的截面在一个数量级，也就是$3×10^{−2}$到 3 mb/sr左右(50MeV，80度)\cite{kerr1997analyzing}。
    
    \item \textbf{$1/2^{-}$， T=3/2 ，12.51 MeV态 }：由于能量更高了，所以预期截面会比前一个更小一些。

    \item \textbf{可能存在的$1/2^{+}$， T=3/2 态 }：具体的能量和截面都未知了。
\end{enumerate}

\subsection{焦面探测器}

焦面探测器就是一个三层的气体电离室，通过粒子在各层的能损还原粒子的入射位置等信息，结构见图\ref{fig2}。第一层和第三层用于获取位置信息，较薄，每层需要输出两个信号；第二层是$\Delta$E信息，较厚，仅需要输出一个信号；每层仅需加一端的高压。

\begin{figure}[h]
    \centering
    \includegraphics[width=0.9\linewidth]{焦探结构.jpg}
    \caption{焦面探测器结构示意图}
    \label{fig2}
\end{figure}




\begin{enumerate}


\item 三层焦面探测器的每一层均为以细丝为阳极，第一层和第三层的阳极丝之间的距离为64.5 mm，以1微米(估摸的）厚的镀铝mylar膜作为阴极的气体电离室，在第一层和第三层的阳极丝上下各有一层阴极膜，即总共有四层。从窗膜到第四层阴极膜的距离是7.5 cm。
\item 焦面探测器充入100 Torr的丙烷气体（C3H8,propane）(实验中实际用的P10）
\item 焦面探测器的窗膜材料是四微米的mylar膜，窗的尺寸3.8 cm*120 cm。若使用5微米的kapton膜充放气多次后易漏气，kapton膜裁剪时需要格外小心。\textcolor{red}{另：关大靶室盖前记得取下焦探和PSD的盖板！}

\item 焦面探测器阳极丝的电阻值：第一层13.67$\sim$13.76千欧 ；第二层102$\sim$104千欧；第三层14.58$\sim$14.75千欧。

\item 具体加高压情况：
\begin{itemize}
   \item 第一层：高压1000 V时漏流无；
   \item 第二层：高压1300 V时漏流无；
   \item 第三层：高压1000 V时漏流0.02微安，最高能加到1100 V，此时漏流0.025微安。
\end{itemize}

\item 位置分辨：第一层和第三层的位置分辨理论上为一点几毫米{\kaishu （cite一篇九几年的401的论文，感兴趣的话可问卢灏要论文）} ，实际上使用数字化获取系统时最好分辨为5毫米左右，使用传统前放时为3.1毫米左右。

\item 调整位置分辨可调节数字化获取的参数，通常调整以下几个参数：

{\kaishu 注：ADCinput可在调试时使用，其后面几列是有效的；建议实验时使用ADCinput16，其后面几列无效。} 
\begin{itemize}
   \item EnergyRiseTime:自行调整；
   \item EnergyFlatTime:需要大于最大的上升时间；
   \item PoleZero:一般可固定，只有锗探测器对这个值敏感。
\end{itemize}

\subsection{小靶室}

小靶室用到了两组（共四片）单面硅和三个小Pin作为monitor，其中单面硅用来做$\Delta$E-E测量，小pin做相对测量中的卢瑟福散射归一。每组单面硅由20微米(前放0.1 p)和300微米（前放1 p)的两块单面硅组成。每片单面硅16路信号，每片小Pin有以路信号，故小靶室中共需5组（10片）前放。

\begin{enumerate}

    \item 部分电流的设定值可以通过在桌面的Q3D设场-NRGFromet.xls文件中输入对应设定值获得。

    \item M2Q,M2S,M2O,M2DE 在电脑的小电源控制软件中设置。需要保证小电源的USB盒子插入电脑。
    \item BD2 要在电脑的NetAssist软件中设置电流，具体方法参考桌面的“电源设置注意事项.txt”。随后查看其实际磁场多次调整。需要保证Q3D控制网线接入同一内网，并电脑的IP在192.168.2.x 网段。
    \item BD1 调整完BD2后，调整Trim1的电流来改变BD1的磁场值。
    \item BD1，BD2磁场查看方法：调整右侧第二个机柜的Probe（有两个Probe按钮都要调），等待面板示数稳定即为当前磁场强度。
    \item BD3 磁场查看方法：左右晃找到共振峰。比较复杂。
\end{enumerate}

\section{附录}

\subsection{俄罗斯人前置工作}

为探究作为基态晕核的同位旋相似态的激发态核中是否还保留了晕结构，俄罗斯人通过$^{10}\text{B}(^{7}\text{Li},^{6}\text{Li})^{11}\text{B}$实验探究了$^{6}\text{Li}$（3.56 MeV 激发态），它与 $^{6}\text{He}$ 和 $^{6}\text{Be}$ 构成 A=6 的同位旋三重态。$^{6}\text{Li}$ 基态可能存在中子晕结构，且可能具有“Tango 结构”，即 $\alpha$ 核心和氘核的准分子结构。其实验结果实验结果支持了$^{6}\text{Li}$（3.56 MeV 激发态）可能存在晕结构的假设。

\subsection{ $^{12}\text{C}(^{6}\text{Li},^{7}\text{Li})^{11}\text{C}$体系的实验调研}

P. Schumacher等人\cite{1973Lithium}在做Li的单核子转移时，发现在其实验中所用的诸多体系里， $^{12}\text{C}(^{6}\text{Li},^{7}\text{Li})^{11}\text{C}$体系的实验结果与DWBA的吻合是最差的，即使是换用不同的势参数体系仍然效果很差。文章猜测是因为该体系的Q值最负以及Li的性质较为复杂。此外，文章认为$^{11}\text{C}$仅有基态，$^{7}\text{Li}$仅分析基态和第一激发态。




P.L Kerr等人\cite{kerr1997analyzing} 对该体系进行了实验研究，主要工作是通过八通道 CCBA 计算描述转移截面和分析功率数据。文献认为出射道的 $^{11}\text{C}$（$J^\pi = 3/2^-$）可以视为仅有基态，同时文章仅分析了 $^{7}\text{Li}$ 的基态（$J^\pi = 3/2^-$）和第一激发态（$J^\pi = 1/2^-$）。
